  --surface-raised: #0a120a;

  --surface-glass: rgba(2,12,2,0.80);

  --green-brand-subtle: rgba(0,255,65,0.08);
  /* Accents, type, spacing, radius, glow… */
}
✓ What this gives you
One file to change a color across the entire casino. No hunting through 43k lines of styles.css. Every future component is built from the same vocabulary — consistent glow, consistent radius, consistent type scale. Provider accent colors become token overrides, not magic numbers.
