💳 Stripe — Card Deposits Checking… ▶
1
Create a free Stripe account at stripe.com — takes 5 minutes. Use a real bank account for payouts. Australia, UK, EU, Canada, NZ are all supported.
2
Go to Dashboard → Developers → API keys.
Copy your Secret key STRIPE_SECRET_KEY (starts with sk_live_)
and Publishable key STRIPE_PUBLISHABLE_KEY (starts with pk_live_).
3
Go to Dashboard → Developers → Webhooks → Add endpoint.
Endpoint URL: https://msaart.online/api/payment/stripe/webhook
Select events: checkout.session.completed, payment_intent.payment_failed, charge.dispute.created, charge.dispute.funds_withdrawn, charge.refunded
Copy the Webhook signing secret → STRIPE_WEBHOOK_SECRET (starts with whsec_).
4
In Render Dashboard: open the matrix-spins service → Environment → Environment Variables. Add these three variables:
STRIPE_SECRET_KEYYour secret key (sk_live_…) STRIPE_PUBLISHABLE_KEYYour publishable key (pk_live_…) STRIPE_WEBHOOK_SECRETWebhook signing secret (whsec_…)
5
Trigger a redeployment in Render Dashboard (service → Manual Deploy → Deploy latest commit).
Card deposits will activate immediately — no code changes needed.
🗄️ PostgreSQL Database Checking… ▶
⚠️ Without DATABASE_URL, the server uses SQLite — which is wiped on every redeploy. All user accounts and balances will be lost. Set up PostgreSQL before taking real money.
1
Create a free PostgreSQL database at Render (or supabase.com). Both have generous free tiers.
2
Copy the connection string. It looks like:
postgresql://user:pass@host/dbname?sslmode=require
3
In Render Dashboard → your matrix-spins service → Environment, add:
DATABASE_URLYour full PostgreSQL connection string
4
Trigger a Manual Deploy in Render Dashboard. The server automatically creates all tables on first boot.
🌐 Custom Domain — msaart.online Action needed ▶
✅ Your site runs on Render.com. Point msaart.online at Render to go live.
1
In Render Dashboard: open the matrix-spins service → Settings → Custom Domains → Add Custom Domain. Enter msaart.online and also www.msaart.online.
2
In your domain registrar (Cloudflare / Namecheap / GoDaddy), update the CNAME record:
TypeCNAME Name@ (apex) or www Valuematrix-spins.onrender.com
If your registrar doesn't support CNAME at apex (@), use an ANAME/ALIAS record instead, or use Cloudflare's proxied CNAME.
3
DNS propagation takes up to 30 minutes. Render auto-provisions an SSL certificate. Your site will be live at https://msaart.online.
